# 飞书工具箱 (Feishu Toolkit)

一个用于飞书 / Lark 网页端的油猴脚本，提供常用的页面增强能力。

本项目是在 3 个上游参考脚本和 1 个早期魔改聚合版的基础上整理而来，当前发布版将这些能力重新命名、模块化，并补上配置面板和安装更新信息。

## 安装

1. 安装 Tampermonkey、Violentmonkey 或其他兼容的 userscript 管理器。
2. 打开脚本安装地址：
   <https://github.com/BlueSkyXN/feishu-toolkit/raw/main/feishu-toolkit.user.js>
3. 确认安装后，访问飞书网页端即可使用。

## 功能

- 去除页面水印图层。
- 解除复制、右键、文本选择限制。
- 可选解除图片 / 文本拖拽限制。
- 保留飞书表格复制到 Excel 时的格式。
- 尝试开启导出、下载、打印等权限位。
- 图片悬停显示下载原图按钮。
- 外链自动使用新标签页打开。
- 使用 `Ctrl+Shift+C` / `Command+Shift+C` 将当前选区复制为 Markdown。
- 通过右侧低干扰入口或油猴菜单打开可视化配置面板。

## 生效时机

| 功能 | 默认 | 生效时机 | 是否需要刷新 |
| --- | --- | --- | --- |
| 去除水印 | 开 | `document-start` 立即注入样式 | 否 |
| 解除复制限制 | 开 | 权限接口返回前改写 XHR 响应 | 是 |
| 解除右键限制 | 开 | 页面早期 hook `addEventListener` | 是 |
| 保留表格格式 | 开 | 复制事件监听注册前决定是否拦截 | 是 |
| 解除文本选择 | 开 | 页面主体出现后注入样式 | 否 |
| 解除拖拽限制 | 关 | 页面主体出现后注入样式 | 否 |
| 强制导出/下载 | 开 | 权限接口返回前改写 XHR 响应 | 是 |
| 图片悬停下载 | 开 | 页面主体出现后绑定悬停事件 | 否 |
| 外链新标签打开 | 开 | 点击链接时判断 | 否 |
| 复制为 Markdown | 开 | 快捷键触发时转换选区 | 否 |
| 设置面板 | 开 | 启动时只挂入口，首次打开时构建完整面板 | 否 |

## 适用范围

脚本匹配以下站点：

- `*.feishu.cn`
- `*.larksuite.com`
- `*.larkoffice.com`

## 说明

脚本会在页面加载早期 hook 部分浏览器 API，并根据飞书网页端当前接口和 DOM 结构做增强处理。飞书前端实现变化后，部分能力可能需要更新适配。

请仅在你有权限访问和处理的文档、图片和页面内容中使用本脚本。

## 配置项说明

| 配置项 | 主要影响 | 前置条件 / 限制 |
| --- | --- | --- |
| 去除水印 | 隐藏当前网页中的常见水印图层，不修改文档内容。 | 依赖飞书水印 DOM 和 CSS 选择器，飞书改版后可能需要补选择器。 |
| 解除复制限制 | 尝试打开权限响应里的复制能力，并优先保留飞书原生富文本复制流程。 | 需要刷新后让 XHR hook 早于权限请求生效；只对网页端返回的权限位有效。 |
| 解除右键限制 | 恢复浏览器原生右键菜单，便于复制、另存图片或检查元素。 | 需要刷新后在页面早期 hook `addEventListener`；已注册的监听不会自动移除。 |
| 解除文本选择 | 强制普通 DOM 文本可选中。 | 对 canvas、图片或特殊编辑器内容不一定生效。 |
| 解除拖拽限制 | 允许拖拽图片和文本。 | 可能改变飞书自己的拖拽交互，默认关闭，需要时再开。 |
| 保留表格格式 | 复制到 Excel / Word 时尽量保留 HTML 表格格式。 | 需要刷新后影响后续 `copy` 监听注册；建议和“解除复制限制”一起开启。 |
| 强制导出/下载 | 尝试打开下载、导出、打印等权限入口。 | 需要刷新后在权限接口返回前生效；受账号权限、文档权限和服务端校验限制。 |
| 图片悬停下载按钮 | 鼠标悬停图片时显示下载按钮，尝试拉取原图保存。 | 图片必须有浏览器可访问的地址；跨域、权限、懒加载或防盗链可能导致失败。 |
| 外链新标签打开 | 站外链接自动新标签打开，飞书域内链接保持原行为。 | 只处理点击时能解析到 `href` 的链接。 |
| 复制为 Markdown | 使用 `Ctrl+Shift+C` / `Command+Shift+C` 将选区转换为 Markdown 并写入剪贴板。 | 需要先选中内容，并且浏览器允许当前页面在用户操作中写剪贴板。 |
| 调试日志 | 在 DevTools console 输出运行信息。 | 仅建议调试时开启；可能让控制台更吵。 |

## 设计取舍

这个版本基于仓库 `ref/` 目录中的 3 个上游脚本，以及早期魔改聚合版 `ref/Feishu SVIP.js` 做了重新整理。目标不是简单堆功能，而是保留日常高频能力，并减少副作用。

- 复制限制优先通过 XHR 权限响应处理，默认不拦截 `copy` 事件，保留飞书自己的富文本 / 表格写剪贴板逻辑，方便复制到 Excel 或 Word。
- 水印选择器参考了独立去水印脚本的覆盖范围，并改成可开关的样式注入。
- 右键限制使用全局 `EventTarget.prototype.addEventListener` hook，比只处理 `document.body` 更早、更完整。
- 图片能力采用悬停下载原图，不沿用旧脚本中的 `navigator.clipboard.writeImage` 写法。
- 功能控制集中在配置面板中，默认只保留核心限制解除和少量高频增强项。

## 来源结构

- `ref/破解飞书.js`：上游参考脚本之一，主要参考 XHR 权限位改写思路；其不拦截 `copy` 事件的行为也是保留表格格式的关键启发。
- `ref/让你的飞书更好用.js`：上游参考脚本之一，主要参考全局事件 hook 和权限响应改写。
- `ref/移除飞书网页水印.js`：上游参考脚本之一，主要参考水印选择器覆盖范围。
- `ref/Feishu SVIP.js`：早期魔改聚合版，已经把解除限制和去水印能力合并；当前 `feishu-toolkit.user.js` 在这个聚合方向上继续做了命名、模块化、配置面板和发布元信息整理。
